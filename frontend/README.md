This folder will contain the .NET MAUI frontend project.

Suggested next steps:
- Create a .NET MAUI project with `dotnet new maui -n JassTournamentManager.Client`
- Move the generated project into this `frontend/` folder
- Configure API URL and authentication in the MAUI app

Notes:
- .NET MAUI requires a compatible SDK and platform tooling (Android/iOS/Windows/macOS).
- For development you can start with Windows/mac desktop targets before adding mobile.
